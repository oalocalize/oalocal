## 📋 Ways to Contribute / Как можно помочь

### 📖 Glossary Contributions / Дополнения в глоссарий

Found a Japanese gaming term we're missing? Great!

**How to submit:**
1. Open an [Issue](../../issues) with the label `glossary`
2. Include: JP term (kanji), romaji, your proposed EN and RU translations, and a usage example from a real game/VN
3. If you're comfortable with Pull Requests — even better, add the term directly to `glossary/terms.md`

**Format:**
```
| JP Term | Romaji | EN | RU | Context |
|---------|--------|----|----|---------|
| 既読スキップ | kidoku sukippu | skip read text | пропуск прочитанного | Common VN UI feature |
```

### 🎮 Review Requests / Запросы на обзоры

Want us to review a specific VN or indie game? Open an Issue with the label `review-request` and tell us:
- Game title and link (Steam / VNDB / itch.io)
- Why you think it deserves a review
- Any specific angle you'd like us to cover (localization quality, cultural adaptation, design)

### 🔧 Translation Suggestions / Предложения по переводу

If you spot an error or have a better translation variant for an active project:
1. Open an Issue or PR referencing the specific file and line
2. Explain *why* your variant is better — context matters more than opinion
3. If possible, include a source (dictionary, native speaker confirmation, in-game screenshot)

### 🐛 Bug Reports / Баг-репорты

Typos, broken links, formatting issues — open an Issue with the label `bug`.

## 📐 Quality Standards / Стандарты качества

### For translations:
- **No raw machine translation.** AI-assisted drafts are acceptable as starting points, but every line must be human-edited and reviewed before merging.
- **Include translation notes** for any non-obvious decision (cultural adaptation, wordplay, name localization).
- **Preserve character voice.** A shy character should sound shy in Russian too, not just say shy things in neutral prose.
- **Test in context.** If possible, check how the text looks in-game or in a mockup before finalizing.

### For glossary entries:
- Every term must include at least one real usage example from a game or VN.
- Avoid dictionary-only definitions — we care about *how the word is actually used in games*.
- If a term has multiple valid translations depending on context, list them all with notes.

### For reviews:
- Minimum 500 words (EN or RU).
- Must include: game metadata (title, developer, VNDB link, platform, length), personal assessment, at least one concrete example from the game.
- Spoiler-free by default. If spoilers are necessary, use `<details>` tags.

## 🔀 Pull Request Process / Процесс Pull Request

1. Fork the repository
2. Create a branch with a descriptive name: `glossary/add-vn-ui-terms` or `review/eden-schemata-en`
3. Make your changes
4. Submit a PR with a clear description of what you changed and why
5. We'll review within 3–5 days and either merge, request changes, or discuss

**First-time contributors:** Don't worry about getting everything perfect. We'd rather have your rough contribution than no contribution at all. We'll help polish it together.

---

## 🚫 What We Don't Accept / Что мы не принимаем

- Raw machine translation without human editing
- Content that violates game developers' explicit wishes regarding fan translations
- Submissions without attribution or sources
- Disrespectful or dismissive comments about other translation projects
